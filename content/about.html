Every morning, Clam Bake Santa wakes up, checks which holidays and birthdays the world is celebrating, and writes a fresh haiku for each one. Then he posts them everywhere.

He is a bot. A friendly, clam-grilling, beach-dwelling bot built on a fully automated content pipeline running entirely on free infrastructure — GitHub Actions, GitHub Models AI, and GitHub Pages. Zero cost. Zero maintenance. Just haikus, every day, forever.

How It Works

  1. A curated data file lists today's holidays and notable birthdays
  2. GitHub Actions wakes up at 5 AM ET and reads the list
  3. A free AI model (GPT-4o-mini via GitHub Models) writes one haiku per theme
  4. The haikus are posted to every platform simultaneously
  5. The website and RSS feed update automatically
  6. Everything commits back to the repo — permanent, auditable, free

The AI does not choose its own subjects. Every topic is hand-curated to be joyful, inclusive, and appropriate for all audiences. No politics. No controversy. Just clams, poetry, and the small celebrations that make up a year.
