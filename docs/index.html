What is this?

Every morning, GitHub Actions wakes up, reads today's holidays and celebrity birthdays from curated data files, calls a free AI to write three haikus, and publishes the results here — and to nine social platforms — automatically. Zero human intervention. Zero cost.

It's also a demonstration of a reusable, plugin-based automation framework: the AI engine, input sources, and output channels are all independently swappable by editing a single config file.

How it works

📅

Curated data files

Holidays and birthdays are read from hand-curated .txt files. You control every subject the AI writes about.

🤖

Free AI generation

GitHub Models (gpt-4o-mini) generates three haikus per day using the GITHUB_TOKEN — no API key purchase required.

🔌

Plugin architecture

Sources, engines, and adapters are swappable plugins. Add a new output channel without touching framework code.

🛡️

Idempotent pipeline

Haikus are cached by date. Re-runs and adapter retries always use the same poems — no double-posting.

Published to 9 platforms

Each adapter runs independently — one failure never stops the others. Missing credentials cause a silent skip.

Mastodon Bluesky Tumblr Telegram Email list Reddit (r/haiku) WordPress GitHub Pages Discord
